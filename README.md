# AI-Integrated-In-Student-Portal
A simple and clean web application with an integrated AI chatbot.
The dashboard allows users to chat with AI and download their conversation as a file. This project is lightweight and easy to run, perfect for learning, demos, and college projects.

# You just need to change the open-router api key in the file (.env)located in the backend folder 
# Then to start the backent open cmd in backend and paste this command (uvicorn main:app --reload) you can check the backend at http://127.0.0.1:8000/docs

# Then just open the index file and type in the correct password 
# It will take you to the dashboard where you can ask question to chat bot at chat area at bottom
